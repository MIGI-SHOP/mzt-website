# MZT Website

This is the official website for MIGI ZONE TOOLZ (MZT), a premium manufacturer of stainless steel instruments for manicure, pedicure, grooming, and dental care.

## Project Structure

- `index.html` - Main homepage
- `css/style.min.css` - Minified CSS styles
- `js/main.min.js` - Minified JavaScript
- `images/` - Image assets (logo, hero background, product images)
- `favicon.ico` - Website favicon
- `sitemap.xml` - XML sitemap for SEO
- `robots.txt` - Robots file for search engines

## Features

- Responsive design
- Modern UI with gradients and animations
- Product showcase sections
- Contact information
- SEO optimized

## Development

To run locally, open `index.html` in a web browser.

For production, ensure all assets are optimized and minified.
